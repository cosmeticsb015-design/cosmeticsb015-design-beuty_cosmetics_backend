# APIs de Strapi

Los elementos de esta carpeta son enlaces simbólicos hacia `src/features/*`.

No agregues lógica nueva directamente aquí si pertenece a una capacidad de negocio. Ubícala en el dominio correspondiente dentro de `src/features` y conserva el enlace de compatibilidad que Strapi necesita para resolver el UID `api::<nombre>.<nombre>`.
